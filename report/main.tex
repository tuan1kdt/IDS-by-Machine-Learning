\begin{abstract}
Intrusion Detection Systems (IDS) are critical components of modern network security infrastructure, designed to identify malicious activities and policy violations in network traffic. Traditional signature-based IDS struggle to detect novel and zero-day attacks, motivating the adoption of machine learning (ML) techniques for anomaly-based intrusion detection. This paper presents a comparative study of three machine learning algorithms for network intrusion detection: \textbf{Random Forest}, \textbf{Deep Neural Network (DNN)}, and \textbf{XGBoost}. We evaluate these models on the \textbf{CIC-IDS 2017} dataset, a modern benchmark containing realistic network traffic with labeled attack categories including DDoS, brute force, web attacks, and botnet traffic. We describe the complete methodology from data preprocessing and feature engineering to model training and evaluation. Experimental results demonstrate that ensemble methods achieve high detection accuracy while maintaining low false positive rates across multiple attack types. We analyze the trade-offs between detection performance, computational cost, and real-time applicability, providing practical insights for deploying ML-based IDS in production environments.

\keywords{Intrusion Detection System \and Machine Learning \and Deep Learning \and CIC-IDS 2017 \and Network Security \and Random Forest \and XGBoost \and Anomaly Detection}
\end{abstract}
