\subsection{Machine Learning Foundations}
\label{sec:bg-ml}

\subsubsection*{Candidate Model Families.}
We evaluate four learners covering the classical decision-tree and boosting spectrum:
\begin{description}[leftmargin=1.2em,itemsep=2pt,topsep=2pt]
    \item[Random Forest (RF)]~\cite{ref_rf} is an ensemble of decision trees trained on bootstrap samples with feature subsampling at each split. It is robust to outliers and invariant to monotonic feature rescaling.
    \item[XGBoost]~\cite{ref_xgboost} builds an additive ensemble of shallow trees fitted sequentially to the pseudo-residuals of the current model. It delivers state-of-the-art results on tabular data and supports regularised objectives for imbalanced targets.
    \item[LightGBM]~\cite{ref_lightgbm} is a histogram-based gradient boosting framework that grows trees leaf-wise rather than level-wise, reducing training time and memory. Its histogram binning strategy makes it particularly efficient on large feature matrices and sensitive to synthetic SMOTE samples near class boundaries.
    \item[Decision Tree (DT)] is a single unpruned CART tree included as an interpretable lower bound that highlights the added value of ensembling.
\end{description}

\subsubsection*{Train / Test Split.}
We use a \textbf{80/20} stratified split (random seed 42): 80\% for training (including SMOTE augmentation) and 20\% as a held-out test set reported exactly once.

\subsubsection*{Evaluation Metrics.}
Because the class distribution remains heavily skewed after sampling, \textbf{Macro-$F_1$} is the primary model-selection criterion---it weights every class equally regardless of support. We additionally report Weighted-$F_1$, ROC-AUC (macro one-vs-rest), Balanced Accuracy, and MCC. Accuracy alone is misleading: a trivial \texttt{BENIGN}-only classifier achieves $\approx 80\%$ accuracy while detecting zero attacks.

Per-class $F_1$ scores are derived from the standard precision and recall:
\begin{equation}
    F_{1,c} = \frac{2 \cdot \text{Precision}_c \cdot \text{Recall}_c}{\text{Precision}_c + \text{Recall}_c}, \qquad \text{Macro-}F_1 = \frac{1}{C}\sum_{c=1}^{C} F_{1,c}.
\end{equation}
