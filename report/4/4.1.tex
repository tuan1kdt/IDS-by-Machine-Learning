\subsection{Experimental Setup}

\paragraph{Hardware.} All experiments were conducted on a machine with Apple M-series CPU, 16GB RAM, running macOS.

\paragraph{Software.} Python 3.10, scikit-learn 1.3~\cite{ref_sklearn}, TensorFlow 2.14~\cite{ref_tensorflow}, XGBoost 2.0~\cite{ref_xgboost}, pandas 2.1, NumPy 1.26, and imbalanced-learn 0.11 (for SMOTE).

\paragraph{Data preparation.} We used the full CIC-IDS 2017 dataset (approximately 2.8 million flows after cleaning). After preprocessing (Section~\ref{sec:preprocessing}), the final dataset contains approximately 2.3 million flows with 52 features. The data is split 80/20 with stratified sampling. SMOTE is applied only to the training set.

\paragraph{Evaluation protocol.} Each model is trained once on the training set and evaluated on the held-out test set. For the DNN, we reserve 10\% of the training data as a validation set for early stopping. We report per-class and macro-averaged metrics.
